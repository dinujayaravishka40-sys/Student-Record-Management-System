# Student Record Management System

A Java-based console application developed for the Data Structures and Algorithms (ICT 143-2) practical assignment at Uva Wellassa University of Sri Lanka.

## 🚀 Core Features & Data Structures Used
This system demonstrates the practical application of three key data structures and two algorithms:

1. **Linked List:** Used to store persistent student records (ID, Name, GPA) dynamically.
2. **Queue (FIFO):** Handles a student registration waiting list.
3. **Stack (LIFO):** Implements an "Undo" feature to reverse the last student registration.
4. **Linear Search:** Used to search for a specific student using their unique ID.
5. **Bubble Sort:** Sorts and displays student records based on their GPA in descending order.

## 🛠️ How to Run
1. Open Visual Studio Code (VS Code).
2. Ensure the **Extension Pack for Java** is installed.
3. Open the `StudentManagementSystem.java` file.
4. Click **Run** above the `main` method or use the Play button.
5. Interact with the application via the built-in VS Code Terminal.
